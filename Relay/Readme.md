## 회로 꽂는방법
### 센서 : 번호
1. 흰색(+) : 2 (5V)
2. 회색(-) : 6 (GND)
3. 노란색(Signal) : 16 (GPIO23)
4. 초록색(가운데 COM) : 4 (5V)
5. 보라색(NO) : 14 (GND)
